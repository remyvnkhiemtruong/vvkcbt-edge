# Offline fonts (LAN)

Place WOFF2 files here for offline exam sites:

- `BeVietnamPro-Regular.woff2` (400)
- `BeVietnamPro-SemiBold.woff2` (600)
- `BeVietnamPro-Bold.woff2` (700)

Download from [Google Fonts — Be Vietnam Pro](https://fonts.google.com/specimen/Be+Vietnam+Pro) or copy from your asset bundle.

If files are missing, the UI falls back to Segoe UI / system-ui.
